# Sviatoslav Cherenov — Portfolio

Персональний сайт-портфоліо системного архітектора та розробника. Проєкт створено для демонстрації ключових компетенцій у сферах Embedded-розробки, проєктування апаратних рішень, створення кросплатформових інтерфейсів та автоматизації.

---

## 🛠 Технологічний стек сайту

Сайт розроблено з акцентом на максимальну продуктивність, швидкість завантаження та чистоту коду:
* **Framework:** [Astro v7](https://astro.build/) (SSG / Static Site Generation)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Design:** Адаптивна верстка під FullHD (1920px) та мобільні пристрої
* **Deployment:** GitHub Pages + GitHub Actions (CI/CD автоматизація збірки)

---

## 📁 Основні напрямки та проєкти

У портфоліо відображено ключові сфери моєї діяльності:
* **Розробка електроніки та Low-Level коду:** Проєктна робота з мікроконтролерами архітектури ESP32 (ESP-IDF, FreeRTOS), інтеграція IoT-систем та робота з протоколами передачі даних (MQTT).
* **Програмне забезпечення:** Створення кросплатформових мобільних та десктопних застосунків (Flutter), написання парсерів даних, автоматизація процесів та розробка Telegram-ботів.
* **Музика та саунд-продюсування:** Досвід роботи зі звуковою архітектурою, координація ритмічних малюнків та створення треків у Bitwig Studio на базі Linux-оточення (Fedora Workstation / Hyprland).
